# Emstel::Generators

## Usage
bin/rails generate ng_model ModelName
bin/rails generate ng_service

Output: tmp/ng/*

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'emstel-generators', git: 'https://github.com/mtzd/emstel-generators'
```

And then execute:
```bash
$ bundle
```
